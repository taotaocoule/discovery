# Discovery

> Data explorer  
> Input : Data  
> Output : Report

## Dependency

python >= 3.3  
numpy  
scipy   
pandas  
matplotlib  
Jinja  
weasyPrint  
[report website](http://pbpython.com/pdf-reports.html)

## Data Type

时间序列数据：time series data  
截面数据：cross section data  

## Statistics

概览信息表 :  
所有特征的名称，基本统计数据（总数，均值，中位数，方差，最大值，最小值，缺失度，数据类型，unique）

概览信息图 : 
 
所有（unique > 10）特征的直方图（一张大图内套小图，能一眼看清所有特征的大致分布）  
所有（unique < 10）特征的饼图
